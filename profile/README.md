Welcome to the **OpenBeavs** GitHub organization! We are an Oregon State University (OSU) CS 46X Capstone team dedicated to building accessible, intelligent tools like the **GENESIS AI Hub**.

🔗 **Live instance:** [genesis.dev.oregonstate.edu](http://genesis.dev.oregonstate.edu)

## 🚀 About Us

OpenBeavs focuses on creating modern software solutions for the OSU community and beyond. Our capstone projects span full-stack web development, Retrieval-Augmented Generation (RAG) pipelines, and integrating AI into functional, user-facing applications. 

## 📂 Featured Repositories

- 🦫 **[OpenBeavs](https://github.com/OpenBeavs/OpenBeavs)**
  The core OSU OpenBeavs application developed for our CS 46X Capstone.
- 🧠 **[OSU-RAG-Pipeline](https://github.com/OpenBeavs/OSU-RAG-Pipeline)**
  A RAG (Retrieval-Augmented Generation) pipeline built in Python to empower our AI knowledge base.

## 🛠️ Tech Stack & Focus

Our development work prominently features:
- **Languages:** JavaScript, Python
- **AI & Data:** RAG architectures, Python backend pipelines
- **Auth & Integration:** Microsoft SSO integrations
- **Web:** Modern JS frameworks, Open WebUI
